# 商业条款草案

这是一份销售页面和购买说明可用的商业条款草案。正式公开前建议根据实际收款方式和经营主体再审阅。

## 产品内容

白歌的AI讨论组专业版包含：

- macOS 桌面应用安装包。
- Blender 桥接插件。
- 普通对话、AI 讨论、工作流、思维导图、摄影参考和后期项目等基础 AI 工作台功能。
- 面向摄影后期和 Blender 建模的一键 3D 建模工作流。
- 智能 Agent 实时渲染面板。
- 混元3D外部服务接口，用户自行部署兼容服务后对接使用。
- 本地 AI 工作台功能。
- 安装说明和首次使用教程。

## 不包含内容

产品不包含：

- 第三方 AI API Key。
- 第三方 AI 模型调用费用。
- Blender 软件本体。
- 第三方素材授权。
- 云端账号。
- 云端存储。
- PS 辅助功能。
- 内置混元3D模型服务、模型权重、算力或第三方服务费用。
- 对任意生成结果的美术质量保证。

团队或机构使用应单独购买团队授权或联系作者。
