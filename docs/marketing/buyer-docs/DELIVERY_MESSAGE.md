# 交付消息模板

## 专业版交付

感谢购买白歌的AI讨论组专业版。

下载内容：

- macOS Apple Silicon 安装包：发送当前版本 arm64 DMG 下载地址。
- macOS Intel 安装包：发送当前版本 x64 DMG 下载地址。
- 文件校验码：发送 `CHECKSUMS.txt`，用于确认安装包下载完整。
- 安装指南：发送 `INSTALL.md` 对应页面或 PDF。
- 首次 3D 场景教程：发送 `FIRST_SCENE_TUTORIAL.md` 对应页面或 PDF。

使用前请准备：

- Blender。
- 你自己的 AI API Key。
- 可用的第三方 API 余额。

重要说明：

- 本软件本地运行，不附赠 API Key。
- 专业版面向摄影后期和 Blender 建模，包含普通对话、AI 讨论、工作流、思维导图、一键 3D 建模和智能 Agent 实时渲染。
- 贩卖版本不包含 PS 辅助。
- 混元3D保留外部服务接口；如果需要使用，请自行部署兼容服务并遵守对应许可。
- 如果当前安装包尚未经过 Apple Developer ID 签名和公证，首次打开可能需要在 macOS“隐私与安全性”里手动允许。
- 3D 功能需要安装 Blender 插件。
- 第三方模型、素材和输出内容的授权由用户自行确认。
- 遇到安装问题，请发送系统版本、芯片类型、Blender 版本和问题截图。

## 支持版交付

感谢购买白歌的AI讨论组支持版。

你将获得：

- 专业版全部内容。
- 一次安装或配置协助。

请先按安装指南尝试完成安装。如果遇到问题，请发送：

- macOS 版本。
- 芯片类型。
- Blender 版本。
- 当前卡住的步骤。
- 错误截图。
