# 首次 3D 场景教程

目标：用白歌的AI讨论组完成一次从文字想法到 Blender 场景雏形的流程。

## 准备

请先完成：

- 安装桌面 App。
- 配置至少一个可用的 AI API。
- 安装并启用 Blender 插件。
- 在软件里点击“测试连接”并确认成功。

## 推荐首次提示词

第一次不要写复杂场景。可以使用：

> 做一个日式小房间，木地板，窗边一张桌子，桌上有一盏暖色台灯和一本书。整体风格温暖，适合做动画短片的开场场景。

## 方法一：一键 3D 建模流水线

1. 打开“摄影工具”。
2. 进入“一键 3D 建模”。
3. 选择已经配置好的文本模型。
4. 输入场景需求。
5. 点击生成场景描述。
6. 按步骤生成参考图、测算参数和 Blender 代码。
7. 确认后推送到 Blender。
8. 在 Blender 里检查结果。

适合：

- 想按流程一步步生成。
- 想看到场景总结、参考图和脚本中间产物。

## 方法二：智能 Agent 实时渲染

1. 打开“摄影工具”。
2. 进入“智能 Agent 实时渲染”。
3. 确认 Blender 桥接地址。
4. 选择默认 API 和模型。
5. 输入场景需求。
6. 点击开始。
7. 观察实时状态、视口监测、AI 思考过程和代码预览。
8. 等待 Agent 完成或手动停止。

适合：

- 想让 Agent 多轮执行和自检。
- 想看“边做边看边修”的过程。

## 如何判断结果是否正常

正常情况：

- Blender 里出现基础物体。
- 软件里有执行日志。
- 代码预览区域持续更新。
- 进度状态会变化。

异常情况：

- Blender 没任何变化。
- 测试连接失败。
- API 报错。
- 模型长时间不返回。
- 生成脚本报错。

## 调整建议

如果结果太简单：

- 增加具体物体数量。
- 指定材质，如木头、金属、玻璃、布料。
- 指定风格，如低多边形、写实、赛博朋克、日式室内。

如果结果混乱：

- 减少物体数量。
- 先生成房间结构，再单独添加装饰。
- 使用更清晰的空间关系。

更好的提示词格式：

> 场景类型：
> 主体物体：
> 材质：
> 灯光：
> 镜头：
> 风格：
> 不要出现：

## 首次成功标准

第一次使用不追求最终美术质量，只确认完整链路：

- App 能调用 API。
- App 能连接 Blender。
- Blender 能接收脚本。
- 场景能生成基础几何。
- 用户知道下一步怎么修改。
