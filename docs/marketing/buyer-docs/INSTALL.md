# 安装指南

适用产品：白歌的AI讨论组专业版

## 购买前确认

使用摄影后期、3D / Blender 建模功能前，请先确认：

- 你有一台 macOS 电脑。
- 你已安装 Blender。
- 你有自己的 AI API Key。
- 你理解本软件不包含第三方 AI 模型费用。
- 你理解 3D 生成结果需要根据模型能力、提示词和 Blender 环境继续调整。
- 你理解贩卖版本不包含 PS 辅助。
- 你理解混元3D保留外部服务接口，若要使用需自行部署兼容服务并遵守对应许可。

## 选择安装包

根据电脑芯片选择 DMG：

- Apple Silicon 芯片：使用 `白歌的AI讨论组-3.7.0-arm64.dmg`
- Intel 芯片：使用 `白歌的AI讨论组-3.7.0-x64.dmg`
- 如果卖家同时提供了 `CHECKSUMS.txt`，可用它核对下载文件是否完整。

查看芯片方式：

1. 点击 macOS 左上角苹果图标。
2. 选择“关于本机”。
3. 查看芯片或处理器信息。

## 安装桌面 App

1. 打开对应的 DMG 文件。
2. 将应用拖入 Applications 文件夹。
3. 首次打开时，如果 macOS 提示来自未知开发者：
   - 打开“系统设置”。
   - 进入“隐私与安全性”。
   - 找到被拦截的应用。
   - 点击“仍要打开”。

说明：如果当前版本尚未经过 Apple Developer ID 签名和 Apple notarization 公证，macOS 首次打开时可能会拦截。这通常不是安装包损坏，而是 macOS 对未公证应用的安全提示。若你不确定下载文件是否完整，请先向卖家索要 SHA256 校验码或 `CHECKSUMS.txt`。

可选校验方式：

```bash
shasum -a 256 白歌的AI讨论组-3.7.0-arm64.dmg
shasum -a 256 白歌的AI讨论组-3.7.0-x64.dmg
```

输出应与卖家提供的 `CHECKSUMS.txt` 一致。

## 配置 AI API

1. 打开白歌的AI讨论组。
2. 进入“设置”。
3. 添加 API：
   - 名称：自定义名称。
   - Base URL：你的 API 服务地址。
   - API Key：你的 API 密钥。
4. 保存后选择常用模型。

注意：API Key 由用户自己提供。软件不会附赠 API Key，也不承担第三方 API 费用。

## 版本说明

专业版用于摄影后期、本地 AI 创作工作台、Blender 建模和实时渲染工作流。贩卖版本不包含 PS 辅助；混元3D保留外部服务接口，用户自行部署和对接。

## 安装 Blender 插件

3D 功能需要 Blender 插件配合。

1. 打开软件里的“摄影工具”。
2. 进入“一键 3D 建模”或“智能 Agent 实时渲染”。
3. 点击“一键导出 Blender 插件到桌面”。
4. 打开 Blender。
5. 进入 `Edit -> Preferences -> Add-ons`。
6. 点击 `Install...`。
7. 选择桌面上的 `aichat_bridge.zip`。
8. 勾选启用插件。
9. 重启 Blender。

## 测试 Blender 连接

1. 保持 Blender 打开。
2. 在软件里进入 3D 面板。
3. 默认桥接地址通常是 `http://127.0.0.1:9876`。
4. 点击“测试连接”。
5. 看到连接成功后，再开始一键建模或 Agent 实时渲染。

如果连接失败：

- 确认 Blender 插件已启用。
- 确认 Blender 已重启。
- 确认桥接地址没有改错。
- 确认本机防火墙没有阻止本地连接。

## 常见问题

### 软件打不开

先检查 macOS 安全设置。首次打开未签名或未公证应用时，macOS 可能需要手动允许。

### API 调用失败

检查 Base URL、API Key、模型名称和账户余额。第三方 API 服务异常不属于软件本身故障。

### Blender 没反应

先点击“测试连接”。如果失败，重新安装插件并重启 Blender。

### 生成结果不好

AI 生成 3D 场景不是一次性完美成品。建议先用简单场景测试，再逐步增加物体、材质和镜头要求。
