# 退款与支持政策草案

## 支持范围

购买 50 元全量版后，基础支持包括：

- 安装包无法打开的排查。
- Blender 插件安装指导。
- API 配置入口说明。
- 首次 3D 场景教程中的流程问题。

支持不包括：

- 代买 API Key。
- 代付第三方模型费用。
- 第三方 API 服务故障。
- 任意 Blender 复杂定制需求。
- 保证生成特定艺术效果。
- 长期一对一教学。

## 退款条件

建议政策：

> 数字产品发货后原则上不支持无理由退款。若软件在标明支持的 macOS 环境中无法启动，且用户在购买后 7 天内提供截图、系统版本和错误信息，并配合排查后仍无法解决，可申请退款。

不建议退款的情况：

- 用户没有 Blender。
- 用户没有 API Key。
- 第三方 API 余额不足或不可用。
- 生成结果不符合主观审美。
- 用户电脑环境不在支持范围内。
- 用户已经成功下载并使用后，以“不想用了”为理由退款。

## 支持信息模板

用户求助时，请让用户提供：

- macOS 版本。
- 芯片类型：Apple Silicon 或 Intel。
- Blender 版本。
- 软件版本。
- 问题截图。
- 操作步骤。
- 是否能点击“测试连接”成功。
- API 报错文本，注意不要发送完整 API Key。

## API Key 安全提醒

用户反馈问题时，不要让用户发送完整 API Key。

如果截图里出现 API Key，应建议用户立即重新生成密钥。
