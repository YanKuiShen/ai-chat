## Pre-Sale FAQ

### 没有云端可以卖吗？

可以。当前版本按本地桌面软件交付，用户自己配置 API Key。没有云端时不要承诺账号同步、在线存储、自动授权或云端算力。

### 为什么不直接用免费 AI 聊天工具？

免费聊天工具通常只解决“聊天”。这个产品卖的是摄影后期和 Blender 场景工作流：项目整理、参考图、工作流、3D 建模入口、Blender 插件桥接和 Agent 实时渲染。

### 会不会 Blender 才能用？

基础 AI 工作台不要求会 Blender。智能 Agent 实时渲染、一键 3D 建模和 Blender 场景功能建议至少知道 Blender 的基本界面。

### 包含 API Key 吗？

不包含。用户需要自己准备 API Key，并承担第三方 API 费用。

### 包含 PS 辅助吗？

不包含。销售版取消 PS 辅助。

### 混元3D可以用吗？

软件保留混元3D外部服务接口，但不内置模型权重、服务进程或算力。需要用户自行部署兼容服务，并遵守原始许可和用途限制。

### 能保证生成效果吗？

不能保证一次生成完美结果。3D 场景质量取决于模型能力、提示词、Blender 环境、插件状态和用户后续调整。

### 可以公开商用吗？

软件可以作为工具出售。用户生成内容是否能商用，取决于用户使用的模型、API、素材和第三方服务许可，不能由本软件统一保证。

## Stop Selling If

Pause manual sales and fix the issue if:

- Two buyers in a row cannot open the app.
- Two buyers in a row cannot install the Blender bridge.
- Buyer reports suggest the install guide is unclear.
- The delivery link leaks private runtime files.
- A release report shows a hard failure.

Do not pause only because users request more features. Record feature requests and keep selling the current scope honestly.
