## Workflow Section (完整创作链路)

Title:

> 从前期拍摄想法到后期 Blender 落地

Lead:

> 一条完整的本地创作链路，面向摄影师和 Blender 用户：前期把想法与参考理清楚，后期把画面落地。

前期拍摄 · 把想法和参考理清楚

1. AI 讨论：让多个模型围绕这次拍摄 / 后期方向给出不同角度，先把思路聊清楚。
2. 拍摄备忘录：把器材、机位、光线和分镜要点记成清单，也可从历史对话一键总结。
3. 思维大纲：用大纲笔记把讨论结果沉淀成结构化清单，再展开成思维导图。
4. 照片墙（情绪板）：把参考图、灵感图拖进照片墙，按项目归档，给画面定调。
5. 图像分析（反推）：对参考图反推提示词与构图、光线、风格要素，方便复用。

后期落地 · 推到 Blender 渲染

6. 工作流：把修图、提示词、建模流程拆成可复用节点，重复任务一键跑通。
7. 一键 3D 建模：从聊天和想法出发，串起场景总结、参考生成、参数测算到 Blender 代码。
8. Blender 实时渲染：Agent 调用插件执行脚本、拉取视口截图自检，循环修正到场景落地。

## FAQ

Q: 这是云端服务吗？

A: 不是。它是本地运行的软件，数据主要保存在用户电脑上。

Q: 购买后是否包含 API Key？

A: 不包含。用户需要配置自己的 API 服务。

Q: 不会 Blender 可以用吗？

A: 可以使用基础 AI 工作台和摄影工具；3D 功能仍建议懂一点 Blender 基础。首批 50 元包含全部内容，不再拆更高价版本。

Q: 生成的 3D 场景能直接商用吗？

A: 需要看你使用的模型、素材、资产和输出内容的许可。软件不替你保证第三方授权。

Q: Windows 能用吗？

A: 项目支持 Windows 打包规划，但首发建议先以 macOS 为主，稳定后再发布 Windows。
