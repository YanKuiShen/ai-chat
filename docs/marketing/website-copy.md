## Workflow Section

Title:

> 从摄影后期想法到 Blender 场景的本地流水线

Steps:

1. 写下摄影后期、背景搭建或 Blender 建模需求，也可以选择历史聊天。
2. AI 总结场景结构和视觉参考。
3. 生成多角度参考图或场景说明。
4. Agent 调用 Blender 插件执行建模脚本。
5. 实时视口检查，继续修正。
6. 保存脚本、计划和反思记录，方便继续修改。

## FAQ

Q: 这是云端服务吗？

A: 不是。它是本地运行的软件，数据主要保存在用户电脑上。

Q: 购买后是否包含 API Key？

A: 不包含。用户需要配置自己的 API 服务。

Q: 不会 Blender 可以用吗？

A: 可以使用基础 AI 工作台；3D 功能仍建议懂一点 Blender 基础，或者购买支持版。

Q: 生成的 3D 场景能直接商用吗？

A: 需要看你使用的模型、素材、资产和输出内容的许可。软件不替你保证第三方授权。

Q: Windows 能用吗？

A: 项目支持 Windows 打包规划，但首发建议先以 macOS 为主，稳定后再发布 Windows。
