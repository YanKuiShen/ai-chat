## Problem Section

Title:

> AI 聊天很多，但把想法变成 3D 场景仍然很慢

Body:

普通 AI 工具可以写文字、出图、总结资料，但 3D 创作还需要反复做场景拆解、Blender 脚本、材质、资产选择和视口检查。白歌的AI讨论组把这些步骤放进一个本地工作台：你给出想法，工具负责组织流程、调用模型、推送 Blender，并保留每一步的脚本和记录。

## Workflow Section

Title:

> 从想法到 Blender 场景的本地流水线

Steps:

1. 写下场景需求或选择历史聊天。
2. AI 总结场景结构和视觉参考。
3. 生成多角度参考图或场景说明。
4. Agent 调用 Blender 插件执行建模脚本。
5. 实时视口检查，继续修正。
6. 保存脚本、计划和反思记录，方便继续修改。

## Feature Section

Feature: AI Blender Agent

> 多角色 Agent 按计划执行、推送脚本、检查视口、反思修正，适合快速搭建场景雏形。

> 从聊天记录或场景描述出发，串起场景总结、参考生成、参数测算、Blender 代码和拼接渲染。

Feature: 本地多模型 AI 工作台

> 支持配置多个 API 和模型，用普通对话、AI 对战、工作流、思维导图辅助创作。

Feature: 摄影与参考图工具

> 管理情绪板、反推图片提示词、生成拍摄备忘录，为 3D 场景提供视觉依据。

Feature: 模板与资产辅助

> 内置 bpy / bmesh / Geometry Nodes 参考和模板，减少 AI 直接写复杂建模脚本时的翻车概率。

## FAQ

Q: 这是云端服务吗？

A: 不是。它是本地运行的软件，数据主要保存在用户电脑上。

Q: 购买后是否包含 API Key？

A: 不包含。用户需要配置自己的 API 服务。

Q: 不会 Blender 可以用吗？

A: 可以使用基础 AI 工作台；3D 功能仍建议懂一点 Blender 基础，或者购买支持版。

Q: 生成的 3D 场景能直接商用吗？

A: 需要看你使用的模型、素材、资产和输出内容的许可。软件不替你保证第三方授权。

Q: Windows 能用吗？

A: 项目支持 Windows 打包规划，但首发建议先以 macOS 为主，稳定后再发布 Windows。
