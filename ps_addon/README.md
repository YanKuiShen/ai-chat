# AIChat PS Bridge — Photoshop MCP

## 简介
本地 HTTP 服务器，连接 AI 和 Photoshop。AI 通过 MCP 指令操控 PS。

## 启动


## MCP 工具
| 工具 | 说明 |
|------|------|
| create_document | 创建新文档 |
| add_text_layer | 添加文字图层 |
| resize_canvas | 调整画布 |
| export_jpg/png | 导出 |
| duplicate_layer | 复制图层 |
| delete_layer | 删除图层 |
| flatten_image | 拼合 |
| get_document_info | 文档信息 |
